# Media File Search Server

HTTP сервер для поиска и предоставления информации о медиафайлах с фоновым обновлением индекса.

## 📖 Описание

Сервер предоставляет список медиафайлов в формате JSON. Фоновый поток периодически обновляет индекс файлов, обеспечивая актуальность данных.
- HTTP сервер на базе `cpp-httplib`
- Фоновое обновление индекса с настраиваемым интервалом
- Обработка ошибок и валидация входных данных

Паоаметры задаются при запуске:
-<path> - обязательный, задайт путь к директории с файлами, значения поумолчанию нет.
-<interval> - не обязательный, задаёт интервал обновления, поумолчанию 1 секунда.

## 📦 Требования

### Системные требования
- **CMake** 3.16 или выше
- **GCC** 11+ или **Clang** 13+ (поддержка C++20)
- **Linux** или **macOS**

### Зависимости
- [cpp-httplib](https://github.com/yhirose/cpp-httplib) - HTTP сервер
- [nlohmann/json](https://github.com/nlohmann/json) - работа с JSON
- Стандартная библиотека C++20 (для `std::jthread`, `std::stop_token`)
